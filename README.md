# Dia
### Your Empathetic AI Companion

Dia is a privacy-first, on-device AI companion designed for wellness, reflection, and meaningful conversation. Built with cutting-edge browser technology, Dia provides a safe space that listens without judgment.

## Core Features
- **Privacy-First**: All AI processing happens locally on your device. Your conversations never leave your browser.
- **WebGPU Powered**: Leverages the latest browser hardware acceleration for lightning-fast, serverless inference.
- **Empathetic Interaction**: Specifically tuned for emotional support, reflection, and relatable, Gen Z-friendly conversation.
- **Zero Latency**: No server round-trips mean instant responses and a fluid chat experience.
- **Secure**: Authentication via Google, but data remains decentralized and private.

## Tech Stack
- **Framework**: Next.js 15 (React 19)
- **AI Engine**: WebLLM / MediaPipe GenAI (WebGPU)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion

*Dia is an AI companion and not a replacement for professional mental health services.*
